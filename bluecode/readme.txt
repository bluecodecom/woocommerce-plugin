Plugin to integrate the bluecode payment method into WooCommerce.

== Description ==

Mobile payment.
Easy, rewarding, secure.

Bluecode is the mobile payment solution that combines contactless payment via smartphone with digital value-added services and enables payment from merchant and banking apps.

This plugin integrates this payment method into your WooCommerce shop.

Please remember that you need an acquiring contract with bluecode before you can use this plugin.  Please find further information on https://www.bluecode.com.

== Installation ==

1. Log into the backend (administration) section of your Wordpress site.
2. Find the "Plugins" menu item and click on it, then clic on the "Add New" button at the top of the page.
3. Click the "Upload Plugin" button.
4. Select the location of the zip file on your local computer and then click "Install Now". The file is now uploaded and installed.
5. Follow the instructions online and finally activate the plugin.
6. Now for the configuration of your payment plugin, go to "WooCommerce -> Settings -> Payments', find the "bluecode" payment method and click in the "Manage" button next to it.
7. On the configuration screen, enable the plugin and enter your credentials received from bluecode.
8. Select the Sandbox mode checkbox to make initial tests against the bluecode sandbox server.  Once your site and contracts are ready to go live, uncheck this option to use the production servers.
